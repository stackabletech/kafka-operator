= Stackable Operator for Apache Kafka

This is an operator for Kubernetes that can manage https://kafka.apache.org/[Apache Kafka] clusters.

WARNING: This operator only works with images from the https://repo.stackable.tech/#browse/browse:docker:v2%2Fstackable%2Fkafka[Stackable] repository

= Dependencies

== ZooKeeper

Kafka currently requires ZooKeeper for coordination purposes.

NOTE: This will change https://cwiki.apache.org/confluence/display/KAFKA/KIP-500[in the future].

Which means a reference to an existing ZooKeeper ensemble must be provided.
